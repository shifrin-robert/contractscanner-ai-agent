# ContractScanner AI Agent

ContractScanner is an AI-powered contract intelligence agent for legal technology and enterprise compliance.

The agent helps users review contracts, identify risky clauses, summarize obligations, and retrieve relevant examples from the CUAD dataset.

## Project Goals

- Build a contract review AI agent
- Use Retrieval-Augmented Generation
- Use CUAD as the contract dataset
- Compare GPT-4o and GPT-4o-mini
- Track agent runs using MLflow
- Evaluate accuracy, relevance, completeness, and hallucination risk

## Main Deliverables

- Data pipeline notebook
- Agent definition notebook
- Evaluation notebook
- Five traced agent examples
- Business ROI analysis
- Final video presentation

## Dataset

Contract Understanding Atticus Dataset, also known as CUAD.
